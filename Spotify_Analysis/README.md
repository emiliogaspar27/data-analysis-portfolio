# Análise de Dados do Spotify ♪

---

## Visão Geral do Projeto
Este projeto foca em **análise exploratória de dados (EDA)** de um dataset do Spotify, com o objetivo de permitir ao leitor **compreender padrões e tendências nas músicas**, incluindo popularidade, duração e características do áudio.  

A análise foi desenvolvida como um **projeto de portfólio**, seguindo boas práticas usadas por analistas de dados em cenários reais.

---

## Objetivos
- Explorar e apresentar **informações resumidas sobre o dataset**  
- Destacar **tendências e padrões gerais das músicas e artistas**  
- Permitir identificação de **faixas mais animadas e atributos musicais relevantes**  
- Organizar os insights de forma **clara e voltada para compreensão rápida do dataset**  
- Fornecer um **relatório profissional** que resume os principais achados

---

## Dataset
- **Fonte:** Kaggle (Dataset do Spotify)  
- **Formato:** CSV  
- **Descrição:** Contém informações sobre músicas, artistas, popularidade, duração e atributos de áudio (danceability, energy, loudness, speechiness, acousticness, instrumentalness, liveness, valence, tempo)  

> O dataset original está incluído neste repositório para garantir **transparência e reprodutibilidade**.
