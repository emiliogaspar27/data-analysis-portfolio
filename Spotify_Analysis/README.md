#  Análise de Dados do Spotify ♪

---

##  Visão Geral do Projeto
Este projeto foca em **análise exploratória de dados (EDA)** de um dataset do Spotify, com o objetivo de entender padrões relacionados a:  
- **Popularidade das músicas**  
- **Duração das faixas**  
- **Artistas**  
- **Características gerais do áudio**  

A análise foi desenvolvida como um **projeto de portfólio**, seguindo boas práticas usadas por analistas de dados em cenários reais.

---

##  Objetivos
- Explorar a estrutura e a qualidade do dataset do Spotify  
- Calcular estatísticas descritivas (métricas resumidas)  
- Identificar padrões e tendências gerais nos dados  
- Organizar os insights de forma clara e voltada para negócios  
- Produzir um relatório profissional utilizando o Excel  

---

##  Dataset
- **Fonte:** Kaggle (Dataset do Spotify)  
- **Formato:** CSV  
- **Descrição:** Contém informações sobre músicas, artistas, pontuações de popularidade, duração e outras características relacionadas ao áudio  

> O dataset original está incluído neste repositório para garantir **transparência e reprodutibilidade**.

---

- **Python**
  - **Pandas** – biblioteca para manipulação e análise de dados
