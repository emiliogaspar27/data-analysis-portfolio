# Data Analysis Portfolio

Olá, eu sou **Emílio Gaspar**, um **Data Science Enthusiast** atualmente focado em **Data Analysis**.  
Este repositório contém projetos práticos desenvolvidos com **datasets** reais, refletindo meu processo contínuo de aprendizado e experiência prática com dados.

O trabalho apresentado aqui enfatiza **exploração, análise, visualização de dados** e o desenvolvimento de pensamento analítico utilizando Python e ferramentas relacionadas.

---

## Objetivo do Repositório

Este repositório foi criado para:
- Praticar técnicas de **data analysis** com **datasets** reais  
- Aplicar conceitos estatísticos e analíticos em diferentes domínios  
- Desenvolver projetos de dados bem estruturados e documentados  
- Manter prática consistente com **Git** e **GitHub**

## Projetos

### Spotify Data Analysis  
Análise exploratória de dados musicais, incluindo métricas de popularidade e informações de artistas.

### Student Performance Analysis  
Análise do desempenho escolar com foco em estatísticas descritivas e comparação entre grupos.

### Heart Disease Analysis  
Exploração de dados de saúde para identificar padrões e variáveis relevantes.

### Superstore Sales Analysis  
Análise de dados comerciais envolvendo vendas, lucro, descontos e desempenho por região.

### Time Series Forecasting  
Análise de dados temporais com ênfase em identificação de tendências, sazonalidade e previsão.

### Financial Data Analysis  
Análise descritiva de dados financeiros e despesas, gerando insights estruturados.

### Text Analysis (NLP)  
Introdução à análise de dados textuais utilizando técnicas básicas de **Natural Language Processing**.

---

## Estrutura dos Projetos

Cada pasta de projeto geralmente contém:
- O **dataset** utilizado na análise  
- Um ou mais **Jupyter Notebooks** com análises comentadas  
- Um **arquivo Excel** com resultados e resumos estruturados  
- Um **README** descrevendo o contexto do projeto, objetivos e principais insights
